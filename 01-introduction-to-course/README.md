# Introduction to Web Programming Course
In the first meeting of the [Web Programming]((https://github.com/wahyuabrory/pemrograman-web.git)) course, we will cover various aspects including an overview of the course, the course materials, requirements, timeline, and more.

## ⛏️Course Requirement
- Text Editor : Notepad, Notepad++, dll
- IDE : VSC0de (live share) , dll
- Web Server : XAMPP, WAMP, dll
- Database (RDBMS) : MySQL, PostgreSQL, dll
- Updated browser
